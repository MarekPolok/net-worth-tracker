# Net Worth Tracker

A simple web application for tracking personal net worth over time.

This project focuses on manual data entry, clear categorization of assets, and visual reports that show how net worth evolves over time. It is intentionally kept simple and transparent, without external financial integrations.

Built publicly as a learning and iteration-focused project.
